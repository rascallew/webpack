## webpack 性能优化
* 开发环境性能优化
* 生产环境性能优化

### 开发环境性能优化
* 优化打包构建速度
  * HMR  
* 优化调试功能
  * source-map

### 生产环境性能优化
* 优化打包构建速度
  * oneOf
  * babel缓存
  * 多进程打包
  * externals
  * dll
* 优化代码运行的性能
  * 缓存（hash chunkhash contenthash）
  * tree shaking(前提：1.必须使用ES6模块化，2.开启production环境)
  * code split
  * 懒加载/预加载
  * pwa
